# MovieTicketBookingSystem
The Movie Ticket Booking System is a Java Swing desktop application that allows users to book movie tickets easily. Users can enter their name, select a movie, choose the number of tickets and calculate the total ticket price. The system also stores booking history and allows users to view previous bookings.
